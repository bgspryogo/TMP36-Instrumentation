1. install node.js
https://nodejs.org/dist/v24.11.1/node-v24.11.1-x64.msi

2. cek versi/instalasi node.js lewat cmd/powershell 
node --version
npm --version

3. buka cmd/powershell, arahin ke directory file ini
cd C/User/....(ganti aja sama nama directorynya)

4. install serialport & socket.io (package node.js buat baca serial data) 
npm install express socket.io serialport

5. run command node data.js di terminal yg sama buat run file instrumennya
tekan ctrl + c buat berhenti

6. buat buka gui nya, buka http://localhost:3000
